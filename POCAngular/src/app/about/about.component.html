Aplicación utilizada para la realización de pruebas de concepto relacionadas con las nuevas tecnologías más demandadas por el mercado como por ejemplo:

  • Desarrollos front-end con Angular o React.
  • Desarrollos back-end con Java, Node o Python.
  • Frameworks de microservicios tipo SpringBoot.
  • Bases de datos NoSQL como MongoDB, Neo4J o Cassandra.
  • Métodos de escalado y orquestación de microservicios.
  • Tipos de comunicación entre microservicios como la comunicacion asíncrona orientada a mensajes.
  • Herramientas tipo APIManagement, APIGateway, LoadBalancing, ServiceDiscovery o CircuitBreaker.
  • Métodos de autenticación y autorización y otros mecanismos de seguridad para front-end y back-end.
  • Tratamiento y procesamiento de logs distribuidos y métricas de aplicación en tiempo real.
